My Projects

Webpage: The Bookworm

Skills used: Python(Flask), HTML, CSS, Bootstrap, PostgreSQL

This project is a dynamic website made using Python's flask framework. Additionally, PostgreSQL is used to interact with database stored on Heroku. Also, data is retrieved from Goodreads API for updated ratings

The followings are the features of the webpage:

  • Creating and Signing in and out of an account
  • Search for a book using keyword
  • Read details (fetched from database and API) about the book
  • Add reviews and rating, and view others' review and rating
  • API access with /api/ route where the webpage will return a JSON response

Mobile Application: Park n Go

Skills used: Flutter SDK

This project is a dynamic mobile application made using Flutter software development kit (SDK) so that it can be used on both IOS and Andriod mobile platform.

Park n Go is designed to ease drivers' journey planning by providing real-time car park avalibility. Some features are:

  • Locating the destination
    • For current location: Located by GPS
    • For other location: Searched and located by Google API
  • See vacancy of each carpark near the destination
    • The real-time vacany data is provided by goverment API from data.gove.sg
  • Bookmark frequency used destination

Java: Hotel Reservation and Payment System

Skills used: Java

This project is implemented using Java.

The following are the features of the project:

  • Locating the destination Create room service menu that can be updated.
  • Create reservation by collecting details of the guest and set an expiry for the reservation.
  • Guest can check in via reservation/walk-in.
  • Guest can make room service order.
  • Staff can search for guest detail, search for room detail and update them accordingly.
  • Guest can check out and charged their total bill including room bill and room service order bill
  • All the details from point 1-6 will be stored in readable text file so that data will be stored even after closing the console
  • Bookmark frequency used destination
